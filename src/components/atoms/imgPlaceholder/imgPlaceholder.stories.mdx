## Beispiele

**JSX:**

```javascript
<ImgPlaceholder />
<ImgPlaceholder width={200} height={200} />
```

**Gerändertes HTML:**

```html
<img
  src="https://fakeimg.pl/200x200/f5f5f5/999999/?font=bebas&amp;font_size=24&amp;text=Dummy"
  srcset="
    https://fakeimg.pl/200x200/f5f5f5/999999/?font=bebas&amp;font_size=24&amp;text=Dummy              1x,
    https://fakeimg.pl/200x200/f5f5f5/999999/?font=bebas&amp;font_size=24&amp;text=Dummy&amp;retina=1 2x
  "
  alt="Placeholder"
/>
```

**Default Image-URL:**
`https://fakeimg.pl/300x300/f5f5f5/999/?font=bebas&font_size=32&text=Dummy&retina=1`

## How to use

### Größe anpassen

Die Größe oder das Format der Bilder kann mit den Attributen `width` oder `height` individuell angepasst werden. Zusätzlich wird über einen URL-Parameter ein retina-optimiertes Bild generiert, dass die 2-fache Bildgröße beinhaltet.
**Fallback/Default:** Bildformat `300 x 300 Pixel`
**URL:** `https://fakeimg.pl/300x300/`

### Text auf Bild anpassen

Mit dem Attribut `text` kann der auf dem generierten Bild ausgegebene Text frei bestimmt werden. Der Default-URL-Parameter wird ersetzt `&?text=Mein neuer Text"`

**Beispiel:** `<ImgPlaceholder text="Mein neuer Text" />`

### Schriftart auf Bild anpassen (font-family)

Mit dem Attribut `font` kann zwischen 3 verschiedenen Schriftarten für den Text auf dem generierten Bild gewählt werden.

| JSX Attribute-Wert | generierter URL-Parameter |
| ------------------ | ------------------------- |
| bebas              | font=bebas                |
| lobster            | font=lobster              |
| museo              | font=museo                |

### Schriftgröße auf Bild anpassen (font-size)

Mit dem Attribut `fontSize` kann die Schriftgröße für den Text auf dem generierten Bild manuell angepasst werden `<ImgPlaceholder fontSize="12" />`.
Die Generierte Bild-URL wird dann z.B. um den Parameter `&font_size=12` erweitert bzw. der Default-Wert ersetzt.

### Retina-Mode

Der Retina-Mode ist standardmäßig gesetzt und erweitert die Bild-URL um den Parameter `retina=1`. Dieser verdoppelt die angegebene Bildgröße für Retina Displays.

### Farbe anpassen

Mit den Attributen `colorBg` und `colorText` können die jeweiligen Farben für Bild und Bildtext bearbeitet werden. Der Farbcode muss im HEX-Format **OHNE** Raute (#) angegeben werden, da der Wert als String an einen URL-Parameter weiter gegeben wird.

**Beispiel:** `<ImgPlaceholder colorBg="000" colorText="fff" />`
